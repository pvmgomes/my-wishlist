# Product Wishlist & Registry

A minimalist, high-performance static site to track and share desired products. 

## Design Philosophy
- **Colors:** Thyme/Olive Green (#708238) and Off-White (#F5F5F0).
- **Architecture:** Serverless, client-side rendering from a JSON data source.
- **Hosting:** GitHub Pages (Zero cost).

## Tech Stack
- **Data:** `products.json`
- **Frontend:** Vanilla HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6 Fetch API).
- **Deployment:** GitHub Pages.

## Project Structure
```text
├── index.html       # Main entry point & layout
├── style.css        # Custom Olive/Off-White styling
├── script.js       # JSON parsing and DOM injection logic
├── products.json    # The "Database" - Add your items here
└── README.md        # Project documentation
